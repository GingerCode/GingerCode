Bienvenido/a!!

  1. En la ventana de la izquierda puedes escribir en pseudocódigo, y se irá traduciendo a código JavaScript en la ventana de la derecha.
  2. Para ejecutar el código resultante, clica el botón Probar, en la tercera ventana de resultado, y se ejecutará el código.
  3. Como puedes observar en la ventana de la derecha está escrito mostrar "Hola Mundo".
  4. Modifícalo, y escribe: mostrar "Hola!! este es mi primer pseudocódigo!!". Este comando es el que usarás cada vez que quieras mostrar algo por consola: mostrar elemento-a-mostrar.
